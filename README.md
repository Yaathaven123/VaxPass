## VaxPass
Blockchain based vaccination record system

### Build with
- Solidity
- Truffle Framework
- Web3JS
- Metamask
- NodeJS
- ReactJS
- Ganache

### To run the Application
- Run Ganache
- Compile and Migrate truffle contract
    npx truffle compile
    npx truffle migrate
    
- Setup and Run Node
    npm install
    npm start
